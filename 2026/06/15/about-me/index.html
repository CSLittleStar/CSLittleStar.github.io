关于我：个人介绍、项目经历与荣誉

你好，我是计科小菜鸟

欢迎来到我的个人博客。这里会记录我的学习路径、项目实践、技术复盘和阶段性成长，也会把那些零散但重要的想法慢慢整理成文章。

如果你是第一次来到这里，可以先从这篇置顶文章认识我；也可以通过导航栏里的「关于」进入独立介绍页。

个人介绍

我是一个正在计算机方向持续探索的学习者。这个博客既是我的知识仓库，也是我的公开笔记本：一边记录技术学习，一边沉淀项目经验。

目前我主要关注这些方向：

  • 人工智能与深度学习：学习 CNN、RNN、Transformer 等模型思想，并尝试用文章复盘理解过程。
  • 边缘计算与物联网：有关云-边-终端协同的物联网模型构建，通过将应用划分为flops和memory两部分，对应用计算量的智能上传卸载、边端负载均衡等内容进行研究。
  • 元启发算法：从传统的NSGA、MOPSO入手，最新研究为RSA的变体，尝试基于模拟退火算法做深入优化。
  • 计算机视觉：关注图像特征提取、模型训练、识别任务和相关应用场景。
  • Web 与个人站点建设：使用 Hexo、Butterfly 和 GitHub Pages 搭建个人博客，持续优化站点内容和展示体验。
  • AI 工具与效率实践：把 AI 作为学习、写作、编程和资料整理的辅助工具，提升从想法到产出的速度。

我希望自己能保持长期主义：多写、多做、多总结，把每一次学习和项目都转化成可复用的经验。

项目经历

个人博客搭建与主题定制

技术栈：Hexo、Butterfly、GitHub Pages、Git

这是当前博客站点本身的建设项目。项目目标是搭建一个稳定、美观、便于长期维护的个人技术博客，用来承载学习笔记、项目总结和个人成长记录。

主要实践包括：

  • 使用 Hexo 作为静态博客生成器，管理文章、分类、标签和页面。
  • 使用 Butterfly 主题优化站点视觉效果、导航结构、首页封面和文章展示。
  • 通过 GitHub Pages 发布博客，实现代码仓库与线上站点的同步部署。
  • 配置分类、标签、友链、图库和关于页面，让站点结构更完整。

这个项目让我更加熟悉静态站点的构建流程，也让我意识到：一个长期维护的个人博客，不只是展示页面，更是一套持续写作和复盘的系统。

云-边-终端协同物联网任务卸载研究

技术关键词：边缘计算、物联网、任务卸载、负载均衡、云-边-终端协同

这个方向主要围绕物联网应用在云端、边缘端和终端之间的协同计算展开。项目思路是将应用需求拆分为 flops 和 memory 两类资源特征，再结合不同节点的计算能力、存储能力和通信开销，研究更合理的任务上传、卸载和负载分配方式。

计划重点包括：

  • 构建云-边-终端协同场景下的任务模型，描述应用、节点和网络之间的关系。
  • 从计算量、内存占用、传输时延和节点负载等角度设计评价指标。
  • 探索智能卸载策略，让任务能够在本地执行、边缘卸载和云端处理之间做出更合适的选择。
  • 关注边端负载均衡问题，避免部分节点过载导致整体系统效率下降。

这个项目会帮助我把物联网系统建模、优化算法和工程实现连接起来，也为后续研究边缘智能、资源调度和多目标优化打基础。

后续文章：云-边-终端协同物联网任务卸载研究

元启发算法与 RSA 改进优化探索

技术关键词：NSGA、MOPSO、RSA、模拟退火、多目标优化

在优化算法方向，我从 NSGA、MOPSO 等经典元启发算法入手，逐步关注 RSA 及其变体。当前更感兴趣的是结合模拟退火思想，对算法的搜索过程、跳出局部最优能力和收敛表现进行改进。

主要探索包括：

  • 梳理不同元启发算法的基本思想、适用场景和优缺点。
  • 分析 RSA 算法的搜索机制，理解其全局探索和局部开发过程。
  • 尝试引入模拟退火策略，提升算法在复杂搜索空间中的稳定性。
  • 结合边缘计算任务卸载等问题，思考优化算法在真实建模场景中的应用方式。

这个项目目前仍处于持续学习和实验阶段，后续会逐步补充算法原理、伪代码、实验设计和结果对比。

后续文章：元启发算法与 RSA 改进优化探索

人工智能与深度学习学习笔记

技术关键词：CNN、RNN、Transformer

在学习人工智能相关内容时，我会把模型结构、核心概念和自己的理解整理成文章。例如卷积神经网络中的卷积层、激活层、池化层和全连接层，以及后续对 RNN、Transformer 等模型的进一步学习。

主要收获包括：

  • 用自己的语言梳理模型结构，避免只记结论、不理解过程。
  • 把抽象概念拆成可解释的小模块，便于后续复习和扩展。
  • 通过博客记录学习过程，让知识积累形成连续的脉络。

计算机视觉模型训练与识别实践

技术关键词：计算机视觉、图像特征提取、模型训练、识别任务

计算机视觉是我在人工智能方向中会持续关注的应用领域。相比只停留在模型概念上，我也希望通过实际任务理解图像数据、特征提取、模型训练和识别结果评估之间的关系。

计划实践包括：

  • 整理图像分类、目标识别等基础任务的处理流程。
  • 学习数据预处理、特征提取、模型训练和结果评估的方法。
  • 结合 CNN 等模型结构，记录从理论理解到实验实践的过程。
  • 持续积累可复现的小实验，为后续更完整的视觉项目做准备。

这个方向会作为深度学习学习笔记的实践延伸，重点记录我如何把模型知识落到具体图像任务中。

AI 辅助学习与内容创作实践

技术关键词：AI 工具、学习效率、内容整理、项目复盘

我也在尝试把 AI 工具融入日常学习和项目开发中，例如辅助整理资料、生成文章初稿、检查代码思路和优化表达方式。

主要实践包括：

  • 用 AI 辅助拆解复杂知识点，再结合自己的理解进行二次整理。
  • 用 AI 帮助检查文章结构，让技术笔记更清晰、更适合复盘。
  • 在项目实现过程中借助 AI 辅助定位问题、生成方案和改进细节。

这个方向还在持续探索中。我的目标不是让 AI 替代思考，而是把它变成一个高效的学习搭档，让自己有更多精力投入理解、判断和创造。
