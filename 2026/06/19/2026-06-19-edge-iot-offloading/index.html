云-边-终端协同物联网任务卸载研究

项目概览

本项目围绕云-边-终端协同的物联网模型构建展开，关注应用计算量的智能上传卸载、边端资源分配和负载均衡问题。当前设想是将应用需求拆分为 flops 和 memory 两部分，进一步分析任务在终端、边缘节点和云端之间的执行策略。

研究关键词

  • 云-边-终端协同
  • 物联网应用建模
  • flops 与 memory 资源划分
  • 任务卸载策略
  • 边端负载均衡
