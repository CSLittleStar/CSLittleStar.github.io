关于我

你好，我是计科小菜鸟

欢迎来到我的个人博客。这里会记录我的学习路径、项目实践、技术复盘和阶段性成长，也会把那些零散但重要的想法慢慢整理成文章。

我关注计算机基础、人工智能、计算机视觉、Web 开发和工程实践。相比只停留在“看懂”，我更希望把知识真正做成作品：能运行、能复盘、能继续迭代，也能给之后的自己留下一条清晰的成长轨迹。

个人介绍

我是一个正在计算机方向持续探索的学习者。这个博客既是我的知识仓库，也是我的公开笔记本：一边记录技术学习，一边沉淀项目经验。

目前我主要关注这些方向：

  • 人工智能与深度学习：学习 CNN、RNN、Transformer 等模型思想，并尝试用文章复盘理解过程。
  • 计算机视觉：关注图像特征提取、模型训练、识别任务和相关应用场景。
  • Web 与个人站点建设：使用 Hexo、Butterfly 和 GitHub Pages 搭建个人博客，持续优化站点内容和展示体验。
  • AI 工具与效率实践：把 AI 作为学习、写作、编程和资料整理的辅助工具，提升从想法到产出的速度。

我希望自己能保持长期主义：多写、多做、多总结，把每一次学习和项目都转化成可复用的经验。

项目经历

个人博客搭建与主题定制

技术栈：Hexo、Butterfly、GitHub Pages、Git

这是当前博客站点本身的建设项目。项目目标是搭建一个稳定、美观、便于长期维护的个人技术博客，用来承载学习笔记、项目总结和个人成长记录。

主要实践包括：

  • 使用 Hexo 作为静态博客生成器，管理文章、分类、标签和页面。
  • 使用 Butterfly 主题优化站点视觉效果、导航结构、首页封面和文章展示。
  • 通过 GitHub Pages 发布博客，实现代码仓库与线上站点的同步部署。
  • 配置分类、标签、友链、图库和关于页面，让站点结构更完整。

这个项目让我更加熟悉静态站点的构建流程，也让我意识到：一个长期维护的个人博客，不只是展示页面，更是一套持续写作和复盘的系统。

计算机视觉与深度学习学习笔记

技术关键词：CNN、RNN、Transformer、特征提取、神经网络

在学习人工智能相关内容时，我会把模型结构、核心概念和自己的理解整理成文章。例如卷积神经网络中的卷积层、激活层、池化层和全连接层，以及后续对 RNN、Transformer 等模型的进一步学习。

主要收获包括：

  • 用自己的语言梳理模型结构，避免只记结论、不理解过程。
  • 把抽象概念拆成可解释的小模块，便于后续复习和扩展。
  • 通过博客记录学习过程，让知识积累形成连续的脉络。

AI 辅助学习与内容创作实践

技术关键词：AI 工具、学习效率、内容整理、项目复盘

我也在尝试把 AI 工具融入日常学习和项目开发中，例如辅助整理资料、生成文章初稿、检查代码思路和优化表达方式。

主要实践包括：

  • 用 AI 辅助拆解复杂知识点，再结合自己的理解进行二次整理。
  • 用 AI 帮助检查文章结构，让技术笔记更清晰、更适合复盘。
  • 在项目实现过程中借助 AI 辅助定位问题、生成方案和改进细节。

这个方向还在持续探索中。我的目标不是让 AI 替代思考，而是把它变成一个高效的学习搭档，让自己有更多精力投入理解、判断和创造。
